# Initialize MongoDB (https://www.mongodb.com/what-is-mongodb)

-> Linux

    sudo get-apt install mongodb

-> Check version

    mongod --version

-> Database status

    sudo service mongodb status

-> Database start

    sudo service mongodb start

-> Database stop

    sudo service mongodb stop
